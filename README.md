# VOID COURIER - 1차 프로토타입

ZX 스펙트럼풍 운송 로그라이트 슈팅 프로토타입입니다.

## 핵심 루프

1. 구간 정보 확인
2. 분석 자원 투입
3. 화물/무기/사람/모듈 적재
4. 수송선을 직접 조종해 구간 돌파
5. 결과 확인 및 다음 구간 진행

## 조작

- 방향키 / WASD: 이동
- Space: 발사
- Z: 실드 과충전
- X: 플레어/교란
- Enter: 선택 / 진행
- Backspace: 뒤로 / 적재 제거
- R: 재시작

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

## 현재 구현된 것

- 256x192 내부 해상도 기반 ZX풍 화면
- 구간별 정보와 심화 분석
- 적재 슬롯 8칸
- 화물, 탄약, 실드, 피난민, 스캐너, 플레어, 무기 부품
- 4개 고정 구간
- 물량형 적, 사격형 적, 지뢰, 잔해
- 탄약 제한
- 화물칸 손상과 인원 손실
- 간단한 메타 크레딧 결과
